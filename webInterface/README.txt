Interactive web interface for bonesaw 
Use the update list button to get the list of Beaglebone IPs visible to the Shodan api
Enter the IP of one of those Bones in the Get More Data field and hit the button to get a list of open ports and their headers for that IP
Enter the same IP into the Scan field and hit that button to use our scan utility and see what security risks exist
The scan output field will change color based on how high of a threat there is to that Bone's security
Red - Very High
Yellow - Moderately High
Green - No Threat
